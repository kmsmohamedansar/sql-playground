# sql-playground

Browser-based SQL workspace using SQLite in WASM, with helpers that mirror common warehouse-style patterns.

## Goal

Practice SQL in the browser with a small translation layer so habits transfer toward Snowflake-style SQL.

## Stack

Static HTML/JS client, SQLite WASM, no backend required for the demo.

## Run locally

Serve the folder over HTTP (GitHub Pages or any static server). Opening `index.html` directly may block WASM in some browsers.
